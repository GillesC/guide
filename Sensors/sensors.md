# Sensors

| Sensor type   | Number     | Interrupt- or polling-based communication? | 
| ------------- |:-------------:|:-------------:| 
| Sound level sensor     | 3-10 (varies over schools) | Interrupt and polling |
| Environmental sensor      | 0-3 (varies over schools)  |  Interrupt and polling |
